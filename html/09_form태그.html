a태그를 이용해서 서버에 데이터 전송

querystring 방식
url 주소에 특정패턴으로 데이터를 전송
?key=value&key2=value2

네이버검색

form 태그

form태그는 서버에 요청을 보낼때 사용하는 태그
form태그의 자식태그로 input태그를 선언함. input태그에 작성한 값이 서버로 전송

요청방식
GET : default설정으로 데이터를 URL주소에 넣어 전송
POST : 데이터를 body(payload)에 저장하여 전송 * 파일전송할때 반드시 post로 설정
