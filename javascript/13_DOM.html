2. 생성된 요소노드.innerText="추가할 문자열";

생성한 태그 원하는 위치에 배치하기

insertAdjacentElement("위치","배치할 태그") 함수를 이용
위치
형제 : 같은 레벨에 배치
beforebegin : 지정된 태그의 위쪽에 배치
afterend : 지정된 태그의 아래에 배치
자식으로 배치
beforeend : 지정된 태그의 맨 밑에 자식으로 배치 == appendChild()
afterbegin : 지정된 태그의 맨 위에 자식으로 배치
