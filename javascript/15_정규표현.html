정규표현식

특정 형태(패턴)의 문자열을 찾을 때 사용하는 문법

RegExp 객체 -> new new RegExp(정규표현식) 리터럴 -> /정규표현식/flag

검사하는 함수 test() boolean / exec() 문자열

플래그 문자

//g|m|i
g : 전체 검색을 하는 옵션
m : 개행을 새로운 문자열로 인식하는 옵션
i : 대소문자 구분 없이 인식하는 옵션

^,$

시작글자, 끝글자를 설정

^a[bc]{2} -> a로 시작해서 bc가 붙어있는 2글자
.a$ -> a로 끝나는 두글자

.

임의의 한글자
키보드로 작성할 수 있는 모든 문자를 의미
/./ : 아무거나 한 글자 , /../ : 아무거나 두글자
/..강/ : 강으로 끝나는 세글자

[]

[]안에 여러 문자열을 작성, 작성한 문자열을 or로 연결, []는 한글자로 인지
/[abcdefghrj].b/ -> a~j로 시작하고 b로 끝나는 세글자
[a-z] / [A-Z] / [0-9]
